Rede Neural Evolutiva Aplicada a um Simples Joguinho de Carro

Esse projeto visa a implementao, em R, de uma Rede Neural Feedforward com aprendizagem usando algoritmo gentico. A ideia da rede aplicar em uma simulao simples, em que um carro ir desviar de obstculos que aparecem de forma aleatria na pista. O carro pomar a deciso de desviar com base na sada da rede neural.

Esboo da arquitetura da rede Multilayer Perceptron de uma camada oculta com aprendizagem usando algoritmo gentico.
